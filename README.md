# Pig-Dice-Roll-Game

This is fun follow along project from youtube. Pig is a game where you can have between 2 to 4 players. The object is to roll the dice each turn and add up your score to 50. The first to reach 50 wins. You may roll as many times as you like each turn. HOWEVER, if you roll a 1, your turn ends and you do not get points from that turn. You must decide how much you want to roll each turn, as the more you roll the greater chances you have of getting a 1. Points from previous turns where a 1 was not rolled are kept.
